FeedMe v0.1

FeedMe is an easy to use parser for RSS and Atom files. It is based on SimpleRSS, but has some improvements that make it worth considering:

  1. Support for attributes
  2. Support for nested elements
  3. Support for elements that appear multiple times
  4. Syntactic sugar that makes it easier to get at the information you want

One word of caution: FeedMe will be maintained only so long as SimpleRSS does not provide the above features. I will try to keep FeedMe‘s API compatible with SimpleRSS so that it will be easy for users to switch if/when necessary.
